static int com.example.apellido.biometriaandroid.LOGIC.Utilidades.majorToUnisgnedInt ( byte[] bytes)
static

Convierte un array de bytes en un número entero sin signo.

Esta función convierte un array de dos bytes en un número entero sin signo de 16 bits. Si el array no contiene exactamente dos bytes, devuelve 1.

Parameters
bytesArray de bytes que contiene los datos a convertir. Debe tener una longitud de exactamente 2.
Returns
El valor entero sin signo correspondiente a los dos bytes proporcionados. Si el array no tiene longitud 2, devuelve 1.
